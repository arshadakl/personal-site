    About me

    I'm a self-taught MERN stack developer who loves to create dynamic and responsive web applications. I have completed three projects using MongoDB, Express, React, and Node.js, showcasing my skills in both front-end and back-end development. I enjoy learning new technologies and frameworks, and I'm always eager to take on new challenges and improve my coding abilities.

    My journey as a web developer started when I decided to learn HTML, CSS, and JavaScript on my own after finishing high school. I was fascinated by the possibilities of creating interactive and engaging websites using these languages. I wanted to learn more and expand my skills, so I started exploring different frameworks and libraries that could help me build more complex and robust web applications. that's when I discovered the MERN stack, which is a combination of four powerful technologies that work together seamlessly. MongoDB is a NoSQL database that allows me to store and retrieve data in a flexible and scalable way. Express is a web framework that simplifies the process of creating web servers and APIs using Node.js. React is a front-end library that enables me to create user interfaces using components and state management. Node.js is a JavaScript runtime environment that allows me to run JavaScript code on the server-side.

    I am always keen to learn new things and keep myself updated with the latest trends and technologies in the web development field. I am currently learning GraphQL and TypeScript, which are two of the most popular and in-demand technologies in the market. I am looking for new opportunities and challenges that will allow me to use my skills and knowledge to create impactful and meaningful digital solutions. I am also open to collaborating and working with other developers who share the same passion and vision as me.
